# DD-25.1 Phase 3: Rust AST + Classifier — `default` and Nested Destructuring

## Context

Phase 1-2 (JS) established the canonical behavior. Phase 3 extends the Rust AST and classifier to match. Two changes:
1. `TypedParam` gains `default_value: Option<SExpr>` for defaults
2. New `DestructuredField` enum replaces `Vec<TypedParam>` in `DestructuredObject::fields` to support nesting
3. `ArrayParamElement` gains a `Default` variant and a `Nested` variant

## Milestone 2: Add `DestructuredField` enum for nesting

**File**: `crates/lykn-lang/src/ast/surface.rs`

### 2.1 — Define `DestructuredField`

```rust
#[derive(Debug, Clone, PartialEq)]
pub enum DestructuredField {
    /// :type name — flat field (possibly with default)
    Simple(TypedParam),
    /// (alias :type name (object/array ...)) — nested pattern
    Nested {
        alias_name: String,
        alias_name_span: Span,
        type_ann: TypeAnnotation,
        pattern: Box<ParamShape>,
        span: Span,
    },
}
```

### 2.2 — Update `DestructuredObject` to use `DestructuredField`

```rust
DestructuredObject {
    fields: Vec<DestructuredField>,  // was: Vec<TypedParam>
    span: Span,
},
```

### 2.3 — Update `ArrayParamElement` for defaults and nesting

```rust
#[derive(Debug, Clone, PartialEq)]
pub enum ArrayParamElement {
    Typed(TypedParam),
    Rest(TypedParam),
    Skip(Span),
    Nested {                    // NEW: positional nested pattern
        pattern: Box<ParamShape>,
        span: Span,
    },
    NestedWithAlias {           // NEW: aliased nested in array
        alias_name: String,
        alias_name_span: Span,
        type_ann: TypeAnnotation,
        pattern: Box<ParamShape>,
        span: Span,
    },
}
```

Note: `TypedParam` already has `default_value: Option<SExpr>` from Milestone 1, so `ArrayParamElement::Typed(TypedParam { default_value: Some(...), ... })` handles array defaults naturally — no separate `Default` variant needed.

### 2.4 — Update `ParamShape` accessor methods

**`typed_params()`** — recurse into nested patterns:

```rust
pub fn typed_params(&self) -> Vec<&TypedParam> {
    match self {
        Self::Simple(tp) => vec![tp],
        Self::DestructuredObject { fields, .. } => fields
            .iter()
            .flat_map(|f| match f {
                DestructuredField::Simple(tp) => vec![tp],
                DestructuredField::Nested { pattern, .. } => pattern.typed_params(),
            })
            .collect(),
        Self::DestructuredArray { elements, .. } => elements
            .iter()
            .flat_map(|e| match e {
                ArrayParamElement::Typed(tp) | ArrayParamElement::Rest(tp) => vec![tp],
                ArrayParamElement::Skip(_) => vec![],
                ArrayParamElement::Nested { pattern, .. }
                | ArrayParamElement::NestedWithAlias { pattern, .. } => pattern.typed_params(),
            })
            .collect(),
    }
}
```

**`bound_names()`** — include alias names + leaf names:

```rust
pub fn bound_names(&self) -> Vec<&str> {
    match self {
        Self::Simple(tp) => vec![tp.name.as_str()],
        Self::DestructuredObject { fields, .. } => fields
            .iter()
            .flat_map(|f| match f {
                DestructuredField::Simple(tp) => vec![tp.name.as_str()],
                DestructuredField::Nested {
                    alias_name, pattern, ..
                } => {
                    let mut names = vec![alias_name.as_str()];
                    names.extend(pattern.bound_names());
                    names
                }
            })
            .collect(),
        Self::DestructuredArray { elements, .. } => elements
            .iter()
            .flat_map(|e| match e {
                ArrayParamElement::Typed(tp) | ArrayParamElement::Rest(tp) => {
                    vec![tp.name.as_str()]
                }
                ArrayParamElement::Skip(_) => vec![],
                ArrayParamElement::Nested { pattern, .. } => pattern.bound_names(),
                ArrayParamElement::NestedWithAlias {
                    alias_name, pattern, ..
                } => {
                    let mut names = vec![alias_name.as_str()];
                    names.extend(pattern.bound_names());
                    names
                }
            })
            .collect(),
    }
}
```

**`dispatch_type()`** — unchanged. Nested doesn't affect dispatch.

### 3.2 — `parse_object_destructure`: handle `(alias ...)`

Replace the "not yet supported" error (lines ~973-979) with:

```rust
"alias" => {
    // (alias :type name (object/array ...))
    if inner.len() < 4 {
        return Err(err(
            "alias in destructured params requires: (alias :type name (object/array ...))",
            span,
        ));
    }
    match (&inner[1], &inner[2], &inner[3]) {
        (
            SExpr::Keyword { value: type_name, span: kspan },
            SExpr::Atom { value: alias_name, span: nspan },
            SExpr::List { values: pat_values, span: pspan, .. },
        ) => {
            let nested = parse_destructured_param(pat_values, *pspan)?;
            fields.push(DestructuredField::Nested {
                alias_name: alias_name.clone(),
                alias_name_span: *nspan,
                type_ann: TypeAnnotation {
                    name: type_name.clone(),
                    span: *kspan,
                },
                pattern: Box::new(nested),
                span: *lspan, // span of the alias list
            });
        }
        _ => {
            return Err(err(
                "alias: expected (alias :type name (object/array ...))",
                span,
            ));
        }
    }
    i += 1;
}
```

## Milestone 4: Update emitter helpers

**File**: `crates/lykn-lang/src/emitter/forms.rs`

### 4.1 — `param_to_kernel_names`: handle defaults and nesting

```rust
fn param_to_kernel_names(p: &ParamShape) -> Vec<SExpr> {
    match p {
        ParamShape::Simple(tp) => {
            if let Some(ref default_val) = tp.default_value {
                vec![list(vec![atom("default"), atom(&tp.name), default_val.clone()])]
            } else {
                vec![atom(&tp.name)]
            }
        }
        ParamShape::DestructuredObject { fields, .. } => {
            let mut items = vec![atom("object")];
            items.extend(fields.iter().map(|f| match f {
                DestructuredField::Simple(tp) => {
                    if let Some(ref val) = tp.default_value {
                        list(vec![atom("default"), atom(&tp.name), val.clone()])
                    } else {
                        atom(&tp.name)
                    }
                }
                DestructuredField::Nested {
                    alias_name, pattern, ..
                } => {
                    let inner = param_to_kernel_names(pattern);
                    list(vec![atom("alias"), atom(alias_name), inner.into_iter().next().unwrap()])
                }
            }));
            vec![list(items)]
        }
        ParamShape::DestructuredArray { elements, .. } => {
            let mut items = vec![atom("array")];
            items.extend(elements.iter().map(|e| match e {
                ArrayParamElement::Typed(tp) => {
                    if let Some(ref val) = tp.default_value {
                        list(vec![atom("default"), atom(&tp.name), val.clone()])
                    } else {
                        atom(&tp.name)
                    }
                }
                ArrayParamElement::Rest(tp) => list(vec![atom("rest"), atom(&tp.name)]),
                ArrayParamElement::Skip(_) => atom("_"),
                ArrayParamElement::Nested { pattern, .. } => {
                    param_to_kernel_names(pattern).into_iter().next().unwrap()
                }
                ArrayParamElement::NestedWithAlias {
                    alias_name, pattern, ..
                } => {
                    let inner = param_to_kernel_names(pattern);
                    list(vec![atom("alias"), atom(alias_name), inner.into_iter().next().unwrap()])
                }
            }));
            vec![list(items)]
        }
    }
}
```
